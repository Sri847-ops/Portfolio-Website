Music Streaming Service

The Music Management System is a comprehensive Python-based application aimed at facilitating the management and streaming of music. It caters to two primary user types: regular users and artists. Regular users can log in to browse, view, and download available songs.
Artists, on the other hand, can log in to upload new songs to the system, specifying details such as the song name, genre, and language.
The application utilizes a PostgreSQL database to securely store user credentials, song details, and download records, ensuring robust data management.
The graphical user interface (GUI) is developed using Tkinter, providing an intuitive and user-friendly experience.
Upon successful login, users are directed to a window displaying the song list, while artists are taken to an upload window where they can manage their music contributions.

Tech Stacks:

* Programming Language: Python
* Database: PostgreSQL
* GUI Framework: Tkinter

Key Features:


User Authentication:
* Secure login for users and artists.
* Email and password verification with PostgreSQL.

User Roles:
* Users: Browse and download songs.
* Artists: Upload new songs.

Song Management:
* Display song list with details (name, artist, language).
* Bold and large font for song names.
* Download songs with tracking in the database.

Artist Upload Interface:
* Input fields for song name, genre, and language.
* Songs added to the database on upload.

Database Integration:
* PostgreSQL for secure data storage and retrieval.
* Efficient querying and updating of records.

Intuitive GUI:
* User-friendly interface with Tkinter.
* Clear navigation for users and artists.
* Informative messages for actions and errors.
