Resume Maker

The Dynamic Resume Generator is a web application built using Flask that allows users to create and customize resumes online.
Users input personal details, skills, work experience, education, and languages through a web form.
The app dynamically generates a structured HTML resume and provides an option to download it as a PDF.

Key Features:

1) User-friendly form for entering resume details
2) Dynamic HTML resume generation
3) PDF download option for the resume

Technical Stack:

* Backend: Flask (Python)
* Frontend: HTML, CSS, JavaScript
* Templating: Jinja2
* Static file handling for PDF download

How to Use:

1) Fill out the resume form on the homepage.
2) Submit the form to generate and view your resume.
3) Download the resume as a PDF using the provided button.
